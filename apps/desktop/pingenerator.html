PinGenerator
5/16/21
I was having trouble making a pin, So i got the idea to make a random pin generator.

Installation

Installer
1. Download The Files (Click The Download Installer Button)
2. Open The Installer
3. Finish Installing
4. Add PinGenerator to your path
5. Open Cmd
6. Then type in the command "pingen"

Zip
1. Download The Files (Click The Download Zip Button)
2. Extract The Files
3. Add PinGenerator to your path
4. Open Cmd
5. Then type in the command "pingen"

Github
1. Head over to Github (Click The Download Github Button)
2. Download The Files (Click The V Code Button)
3. Extract The Files
4. Add PinGenerator to your path
5. Open Cmd
6. Then type in the command "pingen"
