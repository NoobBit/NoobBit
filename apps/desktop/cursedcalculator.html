Cursed Calculator
6/30/21
Calculators are boring. So I made Cursed Calculator. Its a fun little project I made and It works by Generating a random number between 0 - 100.

Installation

Installer
1. Download The Files (Click The Download Installer Button)
2. Open The Installer
3. Finish Installing
4. Go to /bin/
5. Open CursedCalculator.exe

Zip
1. Download The Files (Click The Download Zip Button)
2. Extract The Files
3. Go to /bin/
4. Open CursedCalculator.exe

Github
1. Head over to Github (Click The Download Github Button)
2. Download The Files (Click The V Code Button)
3. Extract The Files
4. Go to /bin/
5. Open CursedCalculator.exe
