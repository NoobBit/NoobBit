WinTerminal
4/22/21
WinTerminal is a custom command prompt created by: Me. It is Mostly used for developers. Creating files, or checking the time there is tons to do with WinTerminal. WinTerminal was created so The CommandShell (and PowerShell) would not be a headace for developers. Cmd is not a very great program for developers and slows down progress. and Wastes A lot of time and money.

Installation

Installer
1. Download The Files (Click The Download Installer Button)
2. Open The Installer
3. Finish Installing
4. Add WinTerminal to your path
5. Then type in the command "terminal"
6. Then enter the help command to find commands to execute

Zip
1. Download The Files (Click The Download Zip Button)
2. Extract The Files
3. Add WinTerminal to your path
4. Open Cmd
5. Then type in the command "terminal"
6. Then enter the help command to find commands to execute

Github
1. Head over to Github (Click The Download Github Button)
2. Download The Files (Click The V Code Button)
3. Extract The Files
4. Add WinTerminal to your path
5. Open Cmd
6. Then type in the command "terminal"
7. Then enter the help command to find commands to execute
